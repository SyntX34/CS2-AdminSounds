AdminSounds for CS2
AdminSounds is a [CounterStrikeSharp](https://github.com/roflmuffin/CounterStrikeSharp) plugin for CS2 that allows server admins to play sounds easily through a menu system. It supports configurable cooldowns, menu-based control, and automatic config generation.

🔊 Includes official sound workshop support
🛠️ Requires [MultiAddonManager](https://github.com/Source2ZE/MultiAddonManager)

📦 Features
- ✅ Toggle plugin per map (enabled/disabled)

- 🕒 Cooldown timer to prevent sound spam

- 🎛️ Center-screen menu using MenuManager

- ⚙️ Automatically generates config on first load

📥 Installation
- Download and install [MultiAddonManager](https://github.com/Source2ZE/MultiAddonManager) (required dependency)

Start your server once to generate the config file.

Subscribe and install the official sound workshop collection:
🎵 [AdminSounds Workshop Sounds](https://steamcommunity.com/sharedfiles/filedetails/?id=3513535709)

🔧 Configuration
The plugin automatically generates a config file after first run. It supports:

- Plugin toggle per map

- Sound cooldown time

- Menu entries customization

💬 Commands
!adminsounds — Opens the in-game menu.

✅ Requirements
- CS2 server running CounterStrikeSharp

- [MultiAddonManager](https://github.com/Source2ZE/MultiAddonManager) (required)
